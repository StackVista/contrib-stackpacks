## Open Telemetry StackPack

The Open Telemetry StackPack provides out-of-the-box support for traces and metrics ingested via the Open Telemetry protocol. Based on the traces a topology is created from services and service instances and their relations. Components are also created for other resources, like hosts, databases, and Kubernetes resources, that can be derived from the Open Telemetry data.

Next to this also metric bindings and monitors are provided for traces and an increasing number of the languages supported by Open Telemetry like:
* Java
* .NET

You can add custom metric bindings and monitors to support your application-specific use cases via Open Telemetry as well.
